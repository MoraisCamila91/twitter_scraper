## Twitter API Scraper
This repository contains a Python script that utilizes the Twitter API to extract all tweets from a specific user, identified by their user_id. It's important to note that this script requires at least a basic-level account for authentication with the Twitter API.

### Key Features:
- Authentication through the Twitter API for accessing user tweets.
- Complete extraction of tweets from a particular user based on their user_id.
- Support for custom configuration, allowing the specification of the target user and authentication parameters.

### Requirements:
- Basic Twitter account for authentication.
- Python 3.x installed in the execution environment.

### How to Use:
1. Clone this repository to your local environment.
2. Install the necessary dependencies specified in the `requirements.txt` file.
3. Configure your Twitter API credentials in the `keys.py` file.
4. Run the `main.py` script and provide the user_id of the target user when prompted.
5. Tweets will be extracted and saved in a suitable format for further analysis.

### Notes:
- Ensure compliance with the Twitter API's terms of service when using this script.
